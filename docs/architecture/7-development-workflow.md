# 7. Development Workflow

### 7.1 Local Development Setup

1.  Clone repository.
2.  Chạy `pnpm install`.
3.  Tạo file `.env.local` trong `apps/web` và điền cấu hình Firebase.
4.  Chạy `pnpm dev`.

### 7.2 Environment Configuration

File `.env.local` chứa các biến `NEXT_PUBLIC_FIREBASE_*`.
